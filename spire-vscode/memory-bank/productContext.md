# Product Context

## Why This Project Exists
Spire is a VS Code extension that provides an AI-powered coding assistant. It connects to the DeepSeek LLM API and supports the Model Context Protocol (MCP) to integrate external tools (e.g., file operations, trap monitoring).

## User Experience Goals
- Provide a sidebar chat interface for natural language coding assistance
- Support MCP-based tool integration for extensible capabilities
- Maintain a Memory Bank for persistent context across sessions
- Enable multiple workflow strategies (direct, agentic, ReAct)

## Key Features
- Sidebar chat with DeepSeek LLM
- MCP server integration for external tools
- Memory Bank context persistence
- Multiple workflow modes (Agentic, Direct, ReAct)
- File operation tools
- API key configuration via VS Code settings
